2.4GHz vs 5GHz 대역대 간섭 현상 시뮬레이션

📚 WiFi 기본 지식

2.4GHz 대역대

장점: 벽 투과력이 좋고, 장거리 전송에 유리

단점: 속도가 느리고, 다른 기기들과 간섭이 많음

사용 이유: 블루투스, 마이크로웨이브, 무선전화 등과 같은 주파수 대역을 공유하여 일상생활에서 가장 안정적으로 작동

5GHz 대역대

장점: 빠른 전송 속도, 간섭이 적음

단점: 벽 투과력이 약하고, 범위가 좁음

사용 이유: 고속 데이터 전송이 필요한 스트리밍, 게임, 대용량 파일 전송에 최적화

간섭이란?

여러 전자기 신호가 서로 방해를 일으키는 현상

예시: 라디오에서 다른 방송이 들리는 것처럼, WiFi 신호도 다른 기기나 장애물에 의해 방해받을 수 있음

📖 사용 가이드

1단계: 기본 설정

먼저 테스트하고 싶은 WiFi 주파수 대역을 선택하세요. 2.4GHz는 안정적이지만 느리고, 5GHz는 빠르지만 범위가 좁습니다.

2단계: 환경 변수 조정

거리, 벽 개수, 간섭 강도 등을 조정하여 실제 환경을 시뮬레이션해보세요. 각 변수가 신호에 미치는 영향을 관찰할 수 있습니다.

3단계: 실험 실행

"실험 시작" 버튼을 클릭하면 설정된 조건에서 WiFi 신호가 어떻게 변화하는지 실시간으로 확인할 수 있습니다.

4단계: 결과 분석

RSSI(신호 강도), 전송 속도, 신호 품질 등을 통해 현재 설정이 WiFi 성능에 미치는 영향을 분석해보세요.

💡 실험 팁

  • 2.4GHz는 벽 투과력이 좋아 여러 방에서 사용하기 좋습니다
  • 5GHz는 빠른 속도가 필요할 때 사용하세요
  • 채널 1, 6, 11은 서로 간섭이 적습니다
  • 습한 날씨는 WiFi 신호를 약하게 만듭니다
  • 저녁 시간대는 다른 사용자들이 많아 간섭이 증가할 수 있습니다
